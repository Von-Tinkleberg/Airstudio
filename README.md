# AirStudio

AirStudio is a full-featured, no-code, open-source game development software. You can build 2D games for mobile (iOS, Android), desktop and the web. AirStudio is designed to be fast and incredibly intuitive: make games using an easy-to-understand yet powerful event-based system and modular behaviors.
